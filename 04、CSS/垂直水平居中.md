### 三种方式实现 css 垂直水平居中

#### 1、flex 布局

```html
<style>
  .parent {
    margin: 50px;
    border: 1px solid black;
    width: 300px;
    height: 300px;
  }

  .center {
    display: flex;
    justify-content: center;
    align-items: center;
  }

  .son {
    border: 1px solid blue;
    width: 100px;
    height: 60px;
  }
</style>
<div class="parent center">
  <div class="son"></div>
</div>
```

### 2、margin: auto

```html
<style>
  .parent {
    margin: 50px;
    border: 1px solid black;
    width: 300px;
    height: 300px;
    position: relative;
  }

  .center {
    margin: auto;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
  }

  .son {
    position: absolute;
    border: 1px solid blue;
    width: 100px;
    height: 60px;
  }
</style>
<div class="parent">
  <div class="son center"></div>
</div>
```

### 3、transform

```html
<style>
  .parent {
    margin: 50px;
    border: 1px solid black;
    width: 300px;
    height: 300px;
    position: relative;
  }

  .center {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
  }

  .son {
    border: 1px solid blue;
    width: 100px;
    height: 60px;
  }
</style>
<div class="parent">
  <div class="son center"></div>
</div>
```
